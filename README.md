# EduAI - Application d'apprentissage avec IA

Application web pour :
- Résumer des cours PDF avec IA (Gemini)
- Générer des quizz interactifs
- Regarder des tutos YouTube intégrés
- Suivre sa progression (XP, niveau, classement)

## Technologies
- Frontend : Next.js 15 + Tailwind CSS + React
- Backend : Node.js + Express + MySQL (XAMPP)
- IA : Google Gemini
- YouTube : YouTube Data API v3

## Installation locale
1. Clone le repo : `git clone https://github.com/biatenomolas/edu-ai.git`
2. Backend : `cd backend && npm install && npm run dev`
3. Frontend : `cd frontend && npm install && npm run dev`

## Déploiement
- Frontend : Vercel
- Backend : Render
